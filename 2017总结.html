你好，2018

更新于 2018-01-17 | 阅读次数
| 字数统计 1,047 | 阅读时长 4

每到年关，总会看到各种各样的年终总结。大学以来一直坚持着记录一些东西的习惯。趁着今年的政治任务，还是开始了今年的总结。与其说是总结，不如说是好好的给刚刚过去的这段时间说说再见，然后再给新的阶段说句你好。

2017 答应自己的坚持写作， 算是做到了。这一年在博客上一共发表了40篇文章。下半年觉得可以开始输出，所以在开始将后来的文章放在简书上。很幸运被 CocoaChina 的编辑看到了，也算是上过热门了。前两天群里还有人说在百度搜 Codable 还有 safe area 首屏都是我的东西。这也算是一些小成就吧！后来因为一些原因，又跑到了掘金，希望接下来的2018在掘金能混够10w+的阅读1k+的喜欢。

1月看着 《Think Python》 学了一段 python， 以亲身经历映证了一个道理。知识还是要用起来才算是掌握。本来还抱着做 python 的期望，到最后实在是没有应用场景，到最后感觉还是跟没学似的。还是自己 too naive 了。

因为苹果强行屏蔽热更新。一直坚守 OC 的理由突然就没有了。所以开始认认真真的去学习 Swift。很庆幸，这件事情也坚持下来了。正巧 2017 年写的第一篇文章是《Swift 和 OC 混合编程》, 最后一篇文章是《新手应该怎么写 Swift》从刚刚开始入门，到最后自诩老司机。这一年最大的收获恐怕就是这件事情了。

2016年实践了半年的敏捷开发，年初去 thoughtworks 参加了一次技术交流。一直听说 thoughtworks 在敏捷实践上很有经验。2017 也算是对敏捷有过一些思考。从翻译《一个人的敏捷开发》到通读《敏捷革命》。总还是会去思考应该怎么样去提高自己的工作效率。很遗憾，之后经历的公司基本上没有认真在做 Scrum 的。

伴着这一年不断的学习和写做。2017 还有一些成长应该是来自外语这方面。学习 Swift 实在是缺少优秀的中文资料。所以在学习的初期，被迫的花了很多的时间去读国外的东西。先是 Medium， 慢慢的接触到了 RayWenderlich、MikeAsh 这些很不错的内容。一年下来，养成了读英文的习惯。读了 RayWenderlich 出的《The Swift Apprentice Second Edition Beginning programming with Swift 3》、《RxSwift Reactive Programming with Swift》两本书。希望2018年能坚持着个习惯吧！

说到读书，这一年还是看过一些书的。《腾讯是怎么长大的》、《一句顶一万句》、《许三观卖血记》、《白鹿原》、《以色列：一个国家的诞生》、《以色列2:在危机中生存》、《钱商》、《哲学家都干了什么》、《佛祖都说了什么》…. 回看起来，自己的兴趣还真是广泛呢。公司传记，文学，政治、经济、哲学、宗教… 也能吹牛自己是一个爱好广泛的人了。

读了些书，应该也要走一些路的。年初跑了趟贵州，实在是没有留下什么深刻的印象。国庆说走就走去了趟若尔盖。算起来还是更喜欢草原沙漠这类比较大的风景。希望2018能有更多几机会去体会更多中人生！

2017年读了很多的书，写了很多的字。学了一门新的计算机语言。写了几个没什么价值的 App。也不知道还做了什么事情了。2017 没能给自己一个满意的答卷，没有活成自己想要的样子，好些年初定下的计划到最后也因为各种各样的原因没能坚持下去。

希望下一年的自己，坚持写作，坚持阅读，坚持学习。能在某些点上有些更深入的思考。很多想做，却没能做的事情希望新的一年能接着做下去。
